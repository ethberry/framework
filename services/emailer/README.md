# Emailer

## Installation

All modules should be installed as part of monorepo flow from root folder

## Configuration

Copy and edit 

```bash
cp -rf .env.sample .env.development
nano .env.development
```

## Start

Start in HMR mode
```bash
npm run start
```

## Test and Code Quality

```bash
npm run lint
npm run test
```

## Usage

Email preview is available on 

```
http://localhost:3003/
```
